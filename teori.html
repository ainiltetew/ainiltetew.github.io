Himpunan

  1. Himpunan

Pengertian

Himpunan adalah kumpulan dari objek-objek yang berbeda yang digunakan untuk mengelompokkan sejumlah objek.

Objek-objek dalam himpunan dapat berupa:

  • Elemen
  • Unsur
  • Anggota

Contoh: A = {1, 2, 3, 4}

Menggambarkan himpunan A terdiri dari 4 anggota/unsur/elemen yaitu 1, 2, 3, dan 4.

Himpunan Besar

Jika sebuah himpunan berukuran besar atau tak terbatas, bisa digambarkan dengan mendaftar sifat yang diperlukan untuk menjadi anggota.

Contoh:

B = { x | x bilangan bulat genap positif }

Karakteristik Himpunan

Well-defined

Sebuah himpunan dikatakan well-defined, jika secara definitif dapat dinyatakan apakah suatu objek merupakan elemen atau bukan elemen dari himpunan tersebut.

Contoh:

S = { beberapa bilangan asli } → bukan well-defined

S = { empat bilangan asli pertama } → well-defined (elemen: 1, 2, 3, 4)

Ekspresi Himpunan

Sebuah himpunan dapat dinyatakan dengan:

  • Menyebutkan sifat-sifatnya
  • Mendaftar elemen-elemennya

Himpunan bilangan prima ≤ 5:

{2, 3, 5} atau {x | x bilangan prima ≤ 5}

Notasi: a ∈ S (a adalah elemen S)

Himpunan kosong: Φ (tidak memiliki elemen)

Penyajian Himpunan

Enumerasi Elemen

Menuliskan semua elemen himpunan di antara dua tanda kurung kurawal.

Contoh:

A = {1, 2, 3, 4}

Simbol-simbol Baku

Simbol Himpunan
P Bilangan bulat positif = {1, 2, 3, ...}
N Bilangan alami (natural) = {1, 2, ...}
Z Bilangan bulat = {..., -2, -1, 0, 1, 2, ...}
Q Bilangan rasional
R Bilangan riil
C Bilangan kompleks

Himpunan Universal: semesta, disimbolkan dengan U.

Notasi Pembentuk Himpunan

Notasi: {x | syarat yang harus dipenuhi oleh x}

  • Bagian di kiri '|' melambangkan elemen himpunan
  • Tanda '|' dibaca "dimana" atau "sedemikian sehingga"
  • Bagian di kanan '|' menunjukkan syarat keanggotaan
  • Tanda ',' dalam syarat keanggotaan dibaca "dan"

Contoh Notasi Pembentuk

Contoh 1: A adalah himpunan bilangan bulat positif < 5

  • A = {x | x adalah bilangan bulat positif yang kecil dari 5}
  • A = {x | x ∈ P, x < 5}
  • A = {1, 2, 3, 4}

Contoh 2: B adalah himpunan bilangan genap positif ≤ 8

  • B = {x | x adalah himpunan genap positif lebih kecil atau sama dari 8}
  • B = {x | x/2 ∈ P, 2 ≤ x ≤ 8}
  • B = {2, 4, 6, 8}

Relasi Himpunan

Himpunan Bagian

A ⊆ B jika setiap elemen A merupakan elemen B.

Himpunan Sama

A = B jika dan hanya jika A ⊆ B dan B ⊆ A.

Contoh: A = {1, 2, 3, 4} dan B = {3, 2, 4, 1} adalah himpunan yang sama.

Kardinalitas

Jumlah elemen dalam himpunan A disebut kardinal dari himpunan A.

Notasi: n(A) atau |A|

Contoh:

B = {x | x bilangan prima < 20}

B = {2, 3, 5, 7, 11, 13, 17, 19}

n(B) = 8

Himpunan Kosong

Himpunan yang tidak memiliki satupun elemen atau himpunan dengan kardinal = 0.

Notasi: Φ atau {}

Contoh:

E = {x | x < x}, maka n(E) = 0

P = {orang Indonesia yang pernah ke bulan}, maka n(P) = 0

Himpunan Ekuivalen

Himpunan A dikatakan ekuivalen dengan himpunan B jika dan hanya jika kardinal dari kedua himpunan tersebut sama.

Notasi: A ~ B ↔ |A| = |B|

Contoh:

A = {1, 3, 5, 7} dan B = {a, b, c, d}

|A| = 4, |B| = 4, maka A ~ B

Himpunan Saling Lepas

Dua himpunan A dan B dikatakan saling lepas (disjoint) jika keduanya tidak memiliki elemen yang sama.

Notasi: A // B

Contoh:

A = {x | x ∈ P, x < 8} dan B = {20, 30, ...}

Maka A // B

Himpunan Kuasa

Disebut juga powerset. Suatu himpunan A yang elemennya merupakan semua himpunan bagian dari A, termasuk himpunan kosong dan himpunan A sendiri.

Notasi: P(A) atau 2^A

Contoh: A = {1, 2}

P(A) = { {}, {1}, {2}, {1, 2} }

n(A) = 4

Operasi Himpunan

Irisan (Intersection)

Himpunan yang setiap elemennya merupakan elemen dari himpunan A dan himpunan B.

Notasi: A ∩ B = {x | x ∈ A dan x ∈ B}

Gabungan (Union)

Himpunan yang setiap anggotanya merupakan anggota himpunan A atau himpunan B.

Notasi: A ∪ B = {x | x ∈ A atau x ∈ B}

Operasi Himpunan Lanjutan

Komplemen

Himpunan yang elemennya merupakan elemen U yang bukan elemen A.

Notasi: Ā = {x | x ∈ U dan x ∉ A}

Selisih

Himpunan yang elemennya merupakan elemen A dan bukan elemen B.

Notasi: A - B = {x | x ∈ A dan x ∉ B} = A ∩ B̄

Beda Setangkup

Himpunan yang elemennya ada pada himpunan A atau B tetapi tidak pada keduanya.

Notasi: A ⊕ B = (A ∪ B) - (A ∩ B) = (A - B) ∪ (B - A)

Perkalian Kartesian

Himpunan yang elemennya semua pasangan berurutan (ordered pairs) yang mungkin terbentuk dengan komponen pertama dari himpunan A dan kedua dari himpunan B.

Notasi: A × B = {(a, b) | a ∈ A dan b ∈ B}

Contoh: C = {1, 2, 3} dan D = {a, b}

C × D = {(1, a), (1, b), (2, a), (2, b), (3, a), (3, b)}

Sifat Operasi Himpunan

  • Hukum Identitas: A ∪ Φ = A, A ∩ U = A
  • Hukum Komplemen: A ∪ Ā = U, A ∩ Ā = Φ
  • Hukum Involusi: (Ā) = A
  • Hukum Null: A ∩ Φ = Φ, A ∪ U = U
  • Hukum Idempoten: A ∪ A = A, A ∩ A = A
  • Hukum Penyerapan: A ∪ (A ∩ B) = A, A ∩ (A ∪ B) = A
  • Hukum De Morgan: A ∪ B = Ā ∩ B̄
