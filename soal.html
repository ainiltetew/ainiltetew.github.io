Himpunan

  1. Himpunan

1. Tuliskan himpunan A yang berisi bilangan genap kurang dari 12.

Kunci Jawaban: A = {2, 4, 6, 8, 10}

2. Diketahui A = {1, 3, 5} dan B = {1, 2, 3, 4, 5}. Tentukan apakah A ⊆ B dan jelaskan.

Kunci Jawaban: Ya, A ⊆ B karena semua elemen A terdapat di B.

3. Tentukan irisan dari A = {2, 4, 6, 8} dan B = {1, 2, 3, 4, 5}.

Kunci Jawaban: A ∩ B = {2, 4}

4. Diketahui U = {1, 2, 3, 4, 5, 6} dan A = {2, 3, 6}. Tentukan komplemen dari A (Ā).

Kunci Jawaban: Ā = {1, 4, 5}

5. Tuliskan himpunan C yang berisi huruf vokal dalam kata "MATEMATIKA".

Kunci Jawaban: C = {A, E, I}

6. Diberikan A = {a, b} dan B = {1, 2, 3}. Tuliskan seluruh anggota dari A × B.

Kunci Jawaban: A × B = {(a,1), (a,2), (a,3), (b,1), (b,2), (b,3)}

7. Tentukan gabungan dari A = {1, 4, 7} dan B = {2, 4, 6, 8}.

Kunci Jawaban: A ∪ B = {1, 2, 4, 6, 7, 8}

8. Jelaskan apa yang dimaksud dengan dua himpunan yang saling lepas dan berikan contohnya.

Kunci Jawaban: Saling lepas = tidak memiliki elemen yang sama. Contoh: {1, 2, 3} dan {4, 5, 6}.

9. Tuliskan himpunan kuasa dari A = {1, 2}.

Kunci Jawaban: P(A) = { {}, {1}, {2}, {1,2} }

10. Jika A = {1, 2, 3} dan B = {3, 4, 5}, tentukan selisih A − B.

Kunci Jawaban: A − B = {1, 2}
