Lokasi & API Jadwal Sholat (Aladhan)

Menggunakan Aladhan API (gratis, metode perhitungan Kemenag Indonesia). Jadwal tahun ini diambil otomatis sekali di awal tahun, dan tersimpan walau offline.

Upload Jadwal Manual (JSON, per bulan)

Upload data jadwal sholat 1 bulan dalam format JSON (lihat template). Setiap awal bulan baru, upload ulang file JSON bulan tersebut supaya jadwalnya tetap update. Kalau kamu punya data dari Excel/CSV, tinggal susun ulang jadi format yang sama seperti template lalu simpan sebagai file .json.
